## Harmonic Layer (Tonic Notes)

> Each location has a tonic note tied to its founding lineage. The note is permanent.
> When combined with the controlling faction's mode, it generates the location's current harmonic field.
> See `gigagen_location_system.md` for full system documentation.

### Tonic Assignments

> **TODO:** Assign tonic notes to all key locations.
> Strategy: tonic comes from founding lineage (12 families = 12 notes).
> The Forno = Giosme lineage territory → Giosme's note = Forno's tonic.
> Multiple locations can share a tonic if built by the same lineage.

### What Tonic Assignment Will Produce

Once assigned, the system automatically calculates:

1. **Location-faction consonance** — which factions naturally control which territories
2. **Character comfort zones** — where each character feels at home (note vs tonic interval)
3. **Conquest difficulty** — how hard it is to hold territory whose tonic doesn't fit your scale
4. **Disputed zone tension** — harmonic clash when two factions overlap in one location
5. **Migration incentives** — characters naturally drift toward consonant locations
